# <center> 前端代码规范

## 介绍

总结前端开发中的规范，从日常工作中积累

## Development

> 请确保你的 Node.js 版本 >= 8.6。

1. git clone 或 download 该项目
2. cd 进入项目目录
3. 若环境中不存在 yarn/npm，则需要先安装yarn

```shell
# 安装依赖包
yarn # 或 npm i

# 开始
yarn dev # 或 npm run dev
```

## Build

```shell
# 构建
yarn build
```
